# COMP 333 Project: End to End Data Analytics Pipeline📊 

## Project Overview
This project implements a complete end to end data analytics and machine learning pipeline as part of the **COMP 333** final project. The objective is to simulate a real-world data science workflow, starting from raw data acquisition and preparation to predictive modeling, unsupervised learning, evaluation, and presentation of actionable insights.
